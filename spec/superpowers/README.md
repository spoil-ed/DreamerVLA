# Spec Superpowers Archive

This directory stores historical/reference implementation plans that were useful while converging DreamerVLA toward the current architecture.

They are not current architecture source of truth.

## How To Use These Files

- Use `spec/README.md`, `00_overview.md` through `05_ray_runtime.md`, and `99_manual_notes.md` for current architecture decisions.
- Use files under `plans/` only as historical context for why a route or test may exist.
- Do not treat unchecked tasks, old commands, or old file maps in these plans as current requirements without comparing them against current code and the main spec files.
- Do not rewrite `spec/99_manual_notes.md` based on these plans.
